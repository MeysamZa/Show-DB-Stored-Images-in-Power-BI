# Show-DB-Stored-Images-in-Power-BI
Read the article (in English) or watch the video (in Persian), on [LinkedIn](https://www.linkedin.com/pulse/how-show-db-stored-images-powerbi-meysam-zahedi-iibkf/)

Have you ever needed to display images stored in your database directly in Power BI?
It sounds simple, but Power BI only supports image URLs, not binary image data.
In this article, I explore three practical methods (all based on SQL Server and Power BI only) to show images that are stored in your DB tables.
I also share how to overcome Power BI’s 32K text limit using Base64 encoding and a bit of T-SQL creativity.
